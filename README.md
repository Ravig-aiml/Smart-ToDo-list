# Smart To-Do List

## Project Overview

This is a simple command-line Smart To-Do List application developed using Python.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* User-friendly menu interface
* Input validation using exception handling

## Technologies Used

* Python 3

## How to Run

1. Save the file as `todo.py`
2. Open terminal
3. Run:

```bash
python todo.py
```
